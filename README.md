# M-Application

A monorepo for the Price Monitoring System and Management Tools.

## Project Structure

- `apps/web-ele`: Vue 3 + Element Plus frontend dashboard.
- `apps/management-server`: NestJS backend management service.
- `apps/data-engine`: (Removed/Pending) Python data crawling engine.
- `apps/backend-mock`: Development mock server.

## Getting Started

### Prerequisites

- Node.js (v20+)
- pnpm (v10+)

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```
